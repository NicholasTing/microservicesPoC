# Microservices PoC

Buildling a sample React Blog webapp with multiservices.

## Getting Started


### Prerequisites

For each of the folders run

```
npm i
```

```
npm run start
```

Then go to localhost:3000 on your local machine

## Built With

* [React]
* [Node]
* [Express]
